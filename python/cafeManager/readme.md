# cafeManager

## Description
**cafeManager** 데이터 기반의 카페 주문 및 재고/매출 통합 관리 시스템 개발. 비즈니스 환경에서 발생하는 주문 재고 차감 매출 분석의 데이터 흐름을 직접 구현을 해보고 자 만듬

SQL과 파이썬의 연동을 한 프로젝트를 만들어 보기 위하여 실행하본 프로젝트

## 프로젝트 목표
* 비즈니스 환경에서 발생하는 주문 재고 차감 매출 분석의 데이터 흐름을 직접 구현을 해보기.
* 기업의 ERP 시스템처럼 마스터 데이터(메뉴, 가격)와 트랜잭션 데이터(주문 내역)가 유기적으로 연동되어 데이터 무결성을 유지하는 시스템을 목표.
* SQL로 분석하여 관리자가 매출 현황을 한눈에 파악할 수 있는 대시보드를 만들어 보고자 함

## 기능
### 사용자
**상품 주문** 카테고리별 메뉴 조회 및 장바구니 담기 기능   
**실시간 재고 연동** 주문 시 재고 부족 상품은 주문 불가   
**주문 데이터 생성** 결제 완료 시 주문 헤더와 상세 테이블에 데이터 적재   
   
### 관리자 
**마스터 데이터 관리** 신규 메뉴 등록, 가격 수정, 재고 입고   
**매출 리포팅** 일별/월별 매출 집계 및 전월 대비 증감률 분석    
잘 팔리는 메뉴 확인   
메뉴별 판매 순위(Best/Worst) 통계 시각화
