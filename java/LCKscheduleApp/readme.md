## 프로젝트 개요
이 프로젝트는 LCK(LoL Champions Korea) 경기 일정을 조회하고 팬들이 소통할 수 있는 데스크톱 애플리케이션입니다.

1. `LCKapp.jar` 파일이 있는 경로로 이동합니다.

2. 아래 명령어를 입력하여 프로그램을 실행합니다.

## 사용 방법 (User Guide)

### 일반 회원
1. **로그인 페이지 이동**: 앱 실행 시 로그인 화면이 나타납니다.
2. **회원가입**: 계정이 없다면 회원가입을 진행합니다.
3. **로그인**: 회원가입 후 생성된 계정으로 로그인합니다.
4. **경기 페이지 출력**: 로그인 성공 시 전체 경기 일정 리스트가 출력됩니다.
5. **상세보기**: 리스트에서 특정 경기를 선택하면 상세 정보를 확인할 수 있습니다.
6. **댓글 작성**: 상세 페이지 하단 입력란에 댓글을 입력하여 등록할 수 있습니다.

### 어드민 (Admin)
어드민 아이디(`admin01`)로 로그인하면 어드민 전용 패널이 출력됩니다.

#### 1. 어드민 댓글 관리
- **전체 댓글 확인창 출력**: 등록된 모든 댓글을 한눈에 볼 수 있습니다.
- **댓글 삭제**: 삭제하고 싶은 댓글을 선택한 후 삭제 버튼을 누르면 해당 댓글이 삭제됩니다.

#### 2. 어드민 경기 관리
- **전체 경기 확인창 출력**: 등록된 모든 경기 일정을 확인할 수 있습니다.
- **경기 수정**:
    1. 수정하고 싶은 경기를 마우스로 선택합니다.
    2. 상단 입력칸에 수정할 정보를 새롭게 입력합니다.
    3. 아래 `수정` 버튼을 클릭하면 경기가 수정됩니다.
